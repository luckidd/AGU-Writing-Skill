# Space Physics Style

Use this self-contained guide for Chinese-English terminology, notation, and claim calibration. Apply this priority: author-defined meaning > manuscript-wide consistency > this guide > general English convention. Preserve an author's established term unless it is physically incorrect or internally inconsistent; flag rather than silently normalize a scientifically meaningful distinction.

## Particle populations and measurements

| English | Preferred Chinese | Guardrail |
|---|---|---|
| radiation belt | 辐射带 | Use “范艾伦辐射带” only when the proper name is intended. |
| energetic / relativistic / ultrarelativistic electrons | 高能 / 相对论 / 超相对论电子 | Do not infer an energy population from an adjective; retain the stated energy range. |
| flux / differential flux / integral flux | 通量 / 微分通量 / 积分通量 | Do not omit the differential-integral distinction or its energy units. |
| count rate / number density | 计数率 /  数密度 | These are not interchangeable with flux. |
| phase-space density | 相空间密度 | Flux is not phase-space density; state invariant coordinates such as `μ`, `K`, and `L*` when supplied. |
| pitch angle / equatorial pitch angle | 投掷角 / 赤道投掷角 | Preserve local versus equatorial pitch angle. |
| pitch-angle distribution | 投掷角分布 | Do not infer anisotropy or a loss cone without the relevant angular coverage. |
| loss cone / bounce loss cone / drift loss cone | 损失锥 / 弹跳损失锥 / 漂移损失锥 | State which loss cone is meant. |
| trapped / quasi-trapped / precipitating | 捕获 / 准捕获 / 沉降 | Define populations by motion, lifetime, or the stated observational proxy. |

## Changes and mechanisms

| English | Preferred Chinese | Guardrail |
|---|---|---|
| enhancement / depletion | 增强 / 减少（或耗减） | Describe the measured change first; these words do not identify a mechanism. |
| dropout | 通量骤降 | A dropout is an observed decrease, not proof of atmospheric loss or magnetopause loss. |
| precipitation | 沉降 | Reserve for atmospheric loss through a loss cone; not every flux decrease is precipitation. |
| injection / transport | 注入 / 输运 | An injection concerns population entry or dispersion; transport requires spatial or phase-space context. |
| acceleration / energization | 加速 / 增能 | Use acceleration for a supported process; use energization for an observed energy gain without overassigning mechanism. |
| radial diffusion / outward radial diffusion | 径向扩散 / 向外径向扩散 | Do not conflate diffusion with convection, injection, or local acceleration. |
| magnetopause shadowing | 磁层顶阴影 | Separate direct shadowing from subsequent outward diffusion and adiabatic effects. |
| adiabatic / nonadiabatic response | 绝热 / 非绝热响应 | Flux change alone does not establish nonadiabatic acceleration or loss. |
| local acceleration / local loss | 局地加速 / 局地损失 | “Local” requires a defensible spatial, temporal, or phase-space diagnostic. |

## Waves, fields, and regions

| English | Preferred Chinese | Guardrail |
|---|---|---|
| whistler-mode chorus | 哨声模合声波 | State band, polarization, and propagation properties only when supplied. |
| plasmaspheric hiss | 等离子体层嘶声 | Do not call all broadband whistler-mode emissions hiss. |
| electromagnetic ion cyclotron (EMIC) wave | 电磁离子回旋波 | Expand at first use; wave occurrence alone does not prove resonant scattering. |
| ultralow-frequency (ULF) wave | 超低频波 | Preserve the stated band and field component. |
| wave-particle / resonant interaction | 波粒 / 共振相互作用 | Require resonance or diffusion diagnostics before claiming effective interaction. |
| magnetic reconnection | 磁重联 | Timing or proximity alone is not proof of reconnection. |
| geomagnetic storm / substorm | 地磁暴 / 磁层亚暴 | Do not use the terms interchangeably; retain the stated index or identification criterion. |
| magnetosphere / ionosphere / thermosphere | 磁层 / 电离层 / 热层 | Preserve region boundaries and coupling direction. |
| magnetopause / bow shock / plasma sheet | 磁层顶 / 弓激波 / 等离子体片 | Preserve boundary, region, and coordinate definitions. |
| plasmasphere / plasmapause | 等离子体层 / 等离子体层顶 | Do not infer the plasmapause from an unspecified density change. |
| solar wind / interplanetary magnetic field | 太阳风 / 行星际磁场 | State the coordinate system for vector components. |

## Geometry, timing, and comparison

| English | Preferred Chinese | Guardrail |
|---|---|---|
| magnetic local time / magnetic latitude | 磁地方时 / 磁纬 | Do not substitute longitude, geographic latitude, or universal time. |
| conjunction / conjugate observation | 交会（或联合）观测 / 共轭观测 | Specify the spatial, temporal, mapping, and tolerance criteria. |
| simultaneous / coincident | 同时 / 时间重合（或共时） | “Coincident” need not mean co-located; state the actual separation. |
| event / interval / episode | 事件 / 时间区间 / 过程 | Use `event` only when selection or physical boundaries are defined. |
| correlation / association / causation | 相关 / 关联 / 因果关系 | Correlation does not establish causation; require timing, mechanism, or an independent diagnostic. |

## Precision rules

- Define abbreviations at first use; preserve official mission and instrument names unless translation is requested.
- Preserve whether a quantity is measured, derived, modeled, fitted, prescribed, or assumed.
- Distinguish energy channel bounds, central energy, integral thresholds, cadence, resolution, and coverage.
- Preserve variables, equations, axes, panel labels, quality flags, coordinate systems, and instrument channels.
- State coordinate systems such as GSM, GSE, SM, geographic, or geomagnetic when they affect interpretation.
- Preserve uncertainty type: measurement uncertainty, variability, confidence interval, model spread, or sensitivity range.
- Use standard units and preserve exponents, including eV/keV/MeV, nT, mV m^-1, Hz/mHz, and spectral-density units.

## Claim verbs

- Direct observation or calculation: `show`, `observe`, `measure`, `reveal`, `yield`.
- Quantified relationship without causality: `vary with`, `correlate with`, `is associated with`.
- Model-data relationship: `reproduce`, `capture`, `constrain`, `estimate`.
- Physical inference: `indicate`, `suggest`, `support`, `is consistent with`.
- Contribution without exclusivity: `contribute to`, `is one factor in`.
- Extrapolation: `may`, `could`, `has the potential to`.
- Avoid `prove`; reserve `demonstrate`, `drive`, `control`, and `cause` for evidence that excludes material alternatives.
